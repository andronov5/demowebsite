About the Club

We meet biweekly on THURSDAYS to learn the basics, collaborate, and build a portfolio you can be proud of.

  • Beginner-friendly workshops that teach by doing.
  • Mentor hours for code reviews and design feedback.
  • Publish with GitHub Pages—free hosting with your own URL.

How we start: super easy. Week 1 is just learning how files work and making a page that says “Good Morning, World!”. Then we add one concept at a time:

HTML = the structure of a page (titles, sections, buttons). CSS = the look (colors, spacing, fonts). JavaScript = the behavior (menus that open, forms that submit, things that react when you click).

Huge emphasis on AI: it’s the future—so we’ll learn to use AI as a teammate to move faster (not something to “beat”). We’ll prompt for layout ideas, generate starter code, and then edit it by hand so you actually learn.

Heads-up about school devices: since AI tools are banned on school computers, we’ll use personal laptops or phones for the AI parts.

Learn the Basics

FAQ

Do I need any experience?

Nope! We teach from the ground up and pair beginners with mentors.

What will I learn?

HTML, CSS, JavaScript, design basics, accessibility, performance, and how to publish.

How much time should I plan?

Bi-weekly lunch meetings, plus optional time outside of school.

Can I use this for college & jobs?

Absolutely! You’ll build a real portfolio and practice talking about your work.
